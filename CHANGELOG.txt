Version 2.0.1
- Updated embedded fdnext to 3caa6f2, expanding managed NAND part-number rules, part-number suggestions, Samsung/SanDisk/Kioxia/SK hynix coverage, and canonical parser metadata keys.
- Added new fdnext NAND decoding rules and bundled resource data, including SMI SATA/NVMe/USB controllers, Yeestor, Maxio, and more.
- Added YMTC/Samsung UFS and eMMC decoding.
- Improved fdnext coverage for Intel, Kioxia BiCS, Western Digital/SanDisk, Phison, and related controller/resource data.
- Improved decode result presentation by reducing repeated part-number fields and promoting Micron FBGA-resolved part numbers in the main result area.

Version 2.0.0
- Migrated the application to Vite, Vue 3, Vuetify 3, and pnpm.
- Added the embedded fdnext parser as the default local decoding engine.
- Kept FlashDetector HTTP API compatibility as an optional server-backed mode.
- Reworked the UI into a denser workstation layout for decoding, searching, settings, About, tables, links, and vendor logos.
- Improved suggestion handling, copy actions, theme and language switching, default input focus, and settings persistence.
- Added a per-major-version ChangeLog dialog that can also be reopened from the About page.
