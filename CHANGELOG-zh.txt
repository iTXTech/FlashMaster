版本 2.1.0
- 更新内嵌 fdnext 到 v2.1.0（6a48b2e），从公开 PN 解析结果中隐藏内部 decodepack token 与分组字段。
- 更新内嵌 fdnext 到 v2.1.0（268679e），适配重命名后的 decodepack 包结构用于内嵌解析器构建。
- 移除旧的 `deploy.bat` GitHub Pages 部署脚本。
- 将 FlashMaster 的产品定位调整为存储芯片智能解析平台（Memory Chip Intelligence Platform），并同步应用文案与文档。
- 重构公开路由为 `/parts`、`/ids`、`/settings`、`/about`，并支持可选 `/en` 与 `/zh` 语言前缀。
- 新增 `VITE_FLASHMASTER_ROUTER_MODE=hash|history`；默认继续使用 hash 模式，history 模式使用正常 URL 并配合 SPA rewrite。
- 更新内嵌 fdnext 到 v2.1.0（4c1c609），扩展 DRAM 与 Managed NAND 覆盖，包含 CXMT DRAM process、Silicon Motion managed NAND、SK hynix ZUFS 4.1 与更完整的旧款 Micron eMCP 解析。
- 简化 About 项目信息，仅强调 GNU AGPL 3.0 许可证，不再重复源码链接和 HTTP API 模式。
- 更新内嵌 fdnext 到 v2.0.0（fe99a8c），改用 fdnext 标准 build metadata，并使用 7 位短 commit hash。
- 设置页解析器/服务器信息现在会从 capabilities API 展示 fdnext 的 commit hash 与 build time，内嵌和 HTTP 模式均支持。
- 新增内置 fdnext HTTP API 预设，包含 iTXTech fdnext 云端与本地 fdnext 开发。
- 更新内嵌 fdnext 到 v2.0.0（c0e005d），新增共享 runtime adapter 层与标准 ExternalLink 结果合约。
- 新增 ExternalLink 渲染，覆盖解析结果与搜索卡片，支持厂商 Logo 链接图，并保留可显式开启的 fdnext processor 预览数据注入。
- 更新内嵌 fdnext 到 v2.0.0（de51be4），对料号搜索中的 marking 候选去重，让 FBGA 与封装标记搜索结果更干净。
- Controller 字段在解析和搜索结果中统一拆分为可折叠列表展示。
- 优化宽屏布局：搜索/设置工作区居中收束，解析详情保持更高信息密度，并精简英文界面文案。
- 更新内嵌 fdnext 到 v2.0.0（9b3ec2f），新增 capabilities inventory 合约，提供解析器、FDB、资源统计和 decoder 元数据。
- 设置页解析器信息改为结构化卡片展示，包含解析器版本、FDB 元数据、资源统计、decoder 列表和支持能力。
- 更新内嵌 fdnext 到 v2.0.0（36fa208），并将所有解析/搜索路径迁移到 fdnext 2.0 标准结果合约。
- 重新基于标准 device、blocks、relations、warnings、candidates 和 search items 渲染 PN 与 Flash ID 结果。
- HTTP 解析模式改为仅支持 fdnext 2.0 接口，并移除外部服务器列表获取。
- 优化宽屏和窄屏下的高密度布局，包括结果头部、字段表格、关联行、搜索卡片和控制器摘要。
- 移除旧的 bit 单位偏好设置，容量/密度显示改由 fdnext 2.0 字段单位统一提供。

版本 2.0.2
- 更新内嵌 fdnext 到 v1.1.0（7176c5b），新增支持DRAM，eMCP，eMMC，UFS，Managed NAND等 PN 解析。
- Managed NAND：扩展 SK hynix、Samsung、SanDisk/KIOXIA、YMTC、Micron 的 PN DSL 覆盖与厂商元数据。
- DRAM：增加 Micron/Crucial、SK hynix、Samsung、Nanya、Elpida、CXMT DRAM 料号解析、资源、文档和测试。
- DRAM 输出：优化公开类型与 CE/Die 分类输出，减少 DRAM 解析结果中的冗余内部字段。
- Micron FBGA/MDB：增加统一 Micron FBGA 资源，通过改进后的抓取刷新 MDB 数据、DRAM 料号搜索数据、原始 MDB 刷新流程，以及内置资源启动修复。
- FDB：增加 FirstChip 控制器生成来源，扩充 FlashDB 控制器与 PN/Flash ID 匹配数据。
- Managed NAND 建议：增加结构化 PN 建议资源，并统一兼容性 baseline 检查。
- 厂商 Logo：补齐 fdnext 当前支持厂商的 Logo 映射，并调整 Kingston、Elpida 等原始商标资源显示比例。

版本 2.0.1
- 更新内嵌 fdnext 到 f870a05，扩展 managed NAND 料号规则、料号建议、三星/闪迪/铠侠/SK hynix 等解析覆盖，统一解析元数据键，并优化 managed NAND 元数据输出。

- 增加 YMTC/Samsung UFS 与 eMMC 解码。
- 优化 fdnext 对 Intel、Kioxia BiCS、Western Digital/SanDisk、Phison 以及相关控制器和资源数据的改进。
- 优化解析结果展示，减少重复料号字段，并将 Micron FBGA 解析出的美光料号提升到主结果区域。
